# TaskFlow

Aplicación web para gestionar tareas de manera local.

## Funcionalidades

- Añadir nuevas tareas con categoría y prioridad.
- Marcar tareas como completadas haciendo click en la tarjeta.
- Borrar tareas individualmente o todas a la vez.
- Filtrar tareas por categoría.
- Buscar tareas por texto.
- Persistencia en LocalStorage, las tareas permanecen al recargar la página.

## Cómo usarlo

1. Añadir tareas en el formulario.
2. Seleccionar categoría y prioridad de cada tarea.
3. Usar el buscador para filtrar tareas por texto.
4. Click en una tarea para marcarla como completada.
5. Click en el botón ❌ para eliminar una tarea.

## Tecnologías

- HTML5
- CSS3
- JavaScript (DOM, LocalStorage)
