# Reflexión final sobre el uso de IA

## Propósito del documento

Este documento recogerá una reflexión final sobre la integración de herramientas de inteligencia artificial en el proceso de desarrollo del proyecto TaskFlow.

Se resumirá:

- Qué aspectos funcionaron bien
- Qué dificultades se encontraron
- Cómo influyó la IA en la productividad
- Qué aprendizajes se obtuvieron
- Recomendaciones para futuros proyectos
